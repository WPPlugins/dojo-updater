== Description ==
This plugin adds an updated [Dojo Toolkit](https://dojotoolkit.org) Javascript library to your set-up.

From 1 October 2016 on, this plugin is no longer supported. Please use [Powerpack](http://www.ramoonus.nl/wordpress/powerpack/) instead.

== Installation ==
1. Upload `dojo-updater/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
